What is it?
============

Consigliere is an AWS Trusted Advisor Dashboard that supports multiple accounts.

Requirements
=============

* [NodeJS](https://nodejs.org/)
* [Redis](https://redis.io/)

Installation
============

```shell
git clone <repo>
npm install
npm start
```

The server will now be available at http://localhost:3000
